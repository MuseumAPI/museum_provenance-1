Overview

Entity is a data structure for holding information about generic entiries used in a Period. It is not currently used on it's own, but as a generic structure to be extended by Party and Location.

- (Boolean) ==(other)

Compare an entity to another entity. Equality is based on both name and certainty.

Examples:

jane_1 = Entity.new("Jane")
jane_2 = Entity.new("Jane")
jane_1 == Jane_2 # -> true

jane_1.certainty = false

def name=(_name) 
  return if _name.nil?
  self.certainty = !detect_uncertainty(_name)
  @name = remove_uncertainty(_name).strip.chomp(",")
end
