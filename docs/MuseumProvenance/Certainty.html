Overview

Extends a class to allow certainty and uncertaintly to be added to the class. it overrides #to_s to add a question mark after the value if the value is uncertain.

Constant Summary

def detect_uncertainty(str)
  is_certain = false
  CertantyWords.each do |certainty_word|
    is_certain = true if str.downcase.include?(certainty_word)
  end
  return is_certain
end

def remove_uncertainty(str)
  phrase = str
  CertantyWords.each do |certainty_word|
    phrase = phrase.gsub(certainty_word, "")
  end
  return phrase.strip
end

def to_s(*args)

  str += CertaintyString unless self.certain?
  str
