Overview

Party is a data structure for holding the associated information about the relevant party of a Period.

A Party is usually a person or organization that is involved with an artwork. It extends Entity with birth and death information, and is intended be extended to include name lookup and biographical functionality.

def name_with_birth_death
  if birth || death
    self.to_s + " [#{birth.to_s(:long) rescue ""}-#{death.to_s(:long) rescue ""}]"
  else
    self.to_s
  end
end
